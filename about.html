ระบบปฏิบัติการเครื่องแม่ข่าย

ข้อมูลเกี่ยวกับระบบปฏิบัติการเครื่องแม่ข่าย...

1. บทนำสู่ระบบปฏิบัติการเครื่องแม่ข่าย

-ความหมายและหน้าที่ของระบบปฏิบัติการเครื่องแม่ข่าย

ความหมายของระบบปฏิบัติการเครื่องแม่ข่าย (Server Operating System)ระบบปฏิบัติการเครื่องแม่ข่ายคือ ซอฟต์แวร์ที่ทำหน้าที่เป็นตัวกลางระหว่างฮาร์ดแวร์ของเซิร์ฟเวอร์และแอปพลิเคชันที่ทำงานบนเครื่องแม่ข่าย โดยระบบปฏิบัติการนี้จะช่วยในการจัดการและควบคุมทรัพยากรต่างๆ เช่น ซีพียู หน่วยความจำ (RAM) ฮาร์ดดิสก์ และเครือข่าย ทำให้เครื่องแม่ข่ายสามารถทำงานและให้บริการได้อย่างมีประสิทธิภาพ

-ประเภทของระบบปฏิบัติการสำหรับเครื่องแม่ข่าย

ประเภทของระบบปฏิบัติการสำหรับเครื่องแม่ข่ายระบบปฏิบัติการสำหรับเครื่องแม่ข่าย (Server Operating Systems) มีหลายประเภท ซึ่งแต่ละประเภทจะเหมาะสมกับการใช้งานที่แตกต่างกัน ขึ้นอยู่กับความต้องการของผู้ใช้งานและประเภทของเซิร์ฟเวอร์ที่ต้องการใช้งาน โดยสามารถแบ่งประเภทหลักๆ ได้ดังนี้:

1. Linux Server

คำอธิบาย: Linux เป็นระบบปฏิบัติการโอเพนซอร์ส (Open Source) ที่ได้รับความนิยมสูงในเครื่องแม่ข่าย เนื่องจากมีความเสถียร, ปลอดภัย, และมีเครื่องมือการจัดการระบบที่มีประสิทธิภาพสูง

2. Windows Server

คำอธิบาย: เป็นระบบปฏิบัติการเซิร์ฟเวอร์ที่พัฒนาโดย Microsoft ซึ่งมักใช้งานในองค์กรที่ต้องการความเข้ากันได้กับซอฟต์แวร์จาก Microsoft หรือในสภาพแวดล้อมที่ใช้ Windows

3. macOS Server

คำอธิบาย: macOS Server เป็นระบบปฏิบัติการที่พัฒนาโดย Apple และใช้สำหรับการบริหารจัดการเครื่องแม่ข่ายที่ใช้ฮาร์ดแวร์ของ Apple

4. Cloud-Based Operating Systems

คำอธิบาย: ระบบปฏิบัติการสำหรับเครื่องแม่ข่ายที่ทำงานในสภาพแวดล้อมของคลาวด์ เช่น ระบบที่ใช้งานผ่านบริการคลาวด์จากผู้ให้บริการต่างๆ เช่น Amazon Web Services (AWS), Microsoft Azure

2. การติดตั้งและตั้งค่าระบบปฏิบัติการเครื่องแม่ข่าย

-ขั้นตอนการติดตั้งระบบปฏิบัติการ

การติดตั้งระบบปฏิบัติการสำหรับเครื่องแม่ข่าย (Server Operating System) เป็นกระบวนการที่ต้องการการวางแผนและการดำเนินการอย่างระมัดระวัง เพื่อให้ระบบสามารถทำงานได้อย่างมีประสิทธิภาพและปลอดภัย นี่คือลำดับขั้นตอนพื้นฐานในการติดตั้งระบบปฏิบัติการเครื่องแม่ข่าย:

1. เตรียมอุปกรณ์: เตรียมเครื่องแม่ข่าย, USB หรือแผ่นติดตั้ง, และเชื่อมต่อเครือข่าย

2. ตั้งค่า BIOS/UEFI: เข้า BIOS/UEFI และตั้งค่าบูตจาก USB หรือ DVD

3. บูตจากแหล่งติดตั้ง: บูตจาก USB หรือ DVD ที่มีไฟล์ติดตั้ง

4. ตั้งค่าภาษาและเวลา: เลือกภาษาและตั้งค่าพื้นฐาน เช่น เวลาและเขตเวลา

5. เลือกประเภทการติดตั้ง: เลือกติดตั้งใหม่หรืออัปเกรด

6. เลือกพาร์ติชัน: สร้างและเลือกพาร์ติชันสำหรับระบบ

7. ติดตั้งระบบปฏิบัติการ: เริ่มการติดตั้งระบบและกำหนดค่าพื้นฐาน

8. ตั้งค่าผู้ใช้: สร้างบัญชีผู้ใช้และตั้งรหัสผ่าน

9. ติดตั้งซอฟต์แวร์เพิ่มเติม: เลือกซอฟต์แวร์ที่จะติดตั้ง (เช่น Apache, MySQL)

10. ตั้งค่าเครือข่าย: กำหนด IP Address และการเชื่อมต่อเครือข่าย

11. รีบูตเครื่อง: หลังการติดตั้งเสร็จ ทำการรีบูตเครื่อง

12. ตรวจสอบการทำงาน: ตรวจสอบการทำงานของระบบและบริการต่างๆ

13. อัปเดตระบบ: อัปเดตระบบและติดตั้งซอฟต์แวร์เพิ่มเติมตามต้องการ

-การตั้งค่าเริ่มต้นของระบบ

การตั้งค่าเริ่มต้นของระบบปฏิบัติการเป็นขั้นตอนสำคัญที่ช่วยให้ระบบพร้อมใช้งานและทำงานได้ตามต้องการ นี่คือลำดับขั้นตอนของการตั้งค่าเริ่มต้น:

1. ตั้งชื่อเครื่อง (Hostname): กำหนดชื่อเครื่องเพื่อระบุในเครือข่าย

2. ตั้งค่าเวลาและเขตเวลา: เลือกเวลาและเขตเวลาให้ตรงกับภูมิภาค

3. ตั้งค่าเครือข่าย: กำหนดที่อยู่ IP และตั้งค่าเครือข่าย

4. ตั้งค่า DNS: กำหนด DNS Servers สำหรับการแปลงชื่อโดเมน

5. สร้างบัญชีผู้ใช้: สร้างบัญชีผู้ใช้และกำหนดสิทธิการเข้าถึง

6. ตั้งค่าความปลอดภัย: เปิดใช้งาน Firewall และตั้งค่า SELinux หรือ AppArmor

7. อัปเดตระบบ: อัปเดตระบบปฏิบัติการให้ทันสมัย

8. ติดตั้งซอฟต์แวร์ที่จำเป็น: ติดตั้งซอฟต์แวร์พื้นฐาน เช่น Apache, MySQL

9. ตั้งค่าการสำรองข้อมูล: ตั้งค่าการสำรองข้อมูลเพื่อป้องกันการสูญหาย

10. ตรวจสอบการทำงานของระบบ: ตรวจสอบสถานะการทำงานของเซอร์วิสต่างๆ

-การตั้งค่าเครือข่ายและ IP Address

1. IP Address แบบ Static: กำหนด IP คงที่

2. IP Address แบบ Dynamic (DHCP): รับ IP อัตโนมัติจาก DHCP

3. Subnet Mask: กำหนด Subnet เช่น 255.255.255.0

4. Gateway: ตั้ง Default Gateway

5. DNS Server: กำหนด DNS Server

6.ทดสอบการเชื่อมต่อ: ใช้คำสั่ง ping ตรวจสอบการเชื่อมต่อ

3. การตั้งค่าและบริหารจัดการบริการในเครื่องแม่ข่าย

-การตั้งค่าและดูแลบริการพื้นฐาน เช่น HTTP, FTP, DNS, DHCP

การตั้งค่าและดูแลบริการพื้นฐาน

1. HTTP: ติดตั้ง Apache หรือ Nginx

2. FTP: ติดตั้ง vsftpd

3. DNS: ติดตั้ง BIND

4. DHCP: ติดตั้ง ISC DHCP Server

-การติดตั้งและตั้งค่าซอฟต์แวร์เซิร์ฟเวอร์ (เช่น Apache, Nginx)

1. Apache: apt install apache2 → systemctl start apache2 → systemctl enable apache2

2. Nginx: apt install nginx → systemctl start nginx → systemctl enable nginx

3. การตั้งค่า: แก้ไขไฟล์การตั้งค่าใน /etc/apache2 (Apache) หรือ /etc/nginx (Nginx) และรีสตาร์ทบริการ

4. ทดสอบ: เปิดเบราว์เซอร์และพิมพ์ http://

4. การดูแลรักษาและตรวจสอบระบบ

-การดูแลและบำรุงรักษาระบบปฏิบัติการ

1.อัปเดตระบบ: ใช้คำสั่ง apt update && apt upgrade (Linux) หรือ "Windows Update"

2. ตรวจสอบระบบ: ใช้คำสั่ง top, htop, systemctl status (Linux) หรือ "Task Manager" (Windows)

3. สำรองข้อมูล: ใช้ rsync, tar (Linux) หรือ "Backup and Restore" (Windows)

4. จัดการ Log Files: ตรวจสอบใน /var/log/ (Linux) หรือ "Event Viewer" (Windows)

5. รักษาความปลอดภัย: ใช้ ufw หรือ iptables (Linux) หรือ "Windows Defender" (Windows)

-การตรวจสอบและวิเคราะห์ประสิทธิภาพของเครื่องแม่ข่าย

1. CPU Usage: ใช้คำสั่ง top หรือ htop (Linux) หรือ "Task Manager" (Windows)

2. Memory Usage: ใช้คำสั่ง free -h (Linux) หรือ "Task Manager" (Windows)

3. Disk Usage: ใช้คำสั่ง df -h (Linux) หรือ "This PC" (Windows)

4. Network Usage: ใช้คำสั่ง netstat หรือ iftop (Linux) หรือ "Resource Monitor" (Windows)

5. Log Files: ตรวจสอบใน /var/log/ (Linux) หรือ "Event Viewer" (Windows)

6. ทดสอบประสิทธิภาพ: ใช้ stress หรือ "Performance Monitor" (Windows)

-การจัดการ Log Files

1. ตรวจสอบ Log: ใช้ cat, less, tail -f (Linux) หรือ "Event Viewer" (Windows)

2. หมุนเวียน Log: ใช้ logrotate (Linux) หรือตั้งค่าใน "Event Viewer" (Windows)

3. เก็บ Log: กำหนดใน /etc/rsyslog.conf (Linux) หรือ "Event Viewer" (Windows)

4. วิเคราะห์ Log: ใช้ grep, awk (Linux) หรือ "Event Viewer" (Windows)

5. สำรอง Log: ใช้ rsync, tar (Linux) หรือ "Backup and Restore" (Windows)

-การตั้งค่าความปลอดภัยพื้นฐาน

1. ใช้รหัสผ่านที่แข็งแรง: รหัสผ่านควรยาวและซับซ้อน รวมทั้งเปิดใช้การยืนยันตัวตนสองขั้นตอน (2FA) เมื่อเป็นไปได้

2. อัปเดตระบบและซอฟต์แวร์: ตรวจสอบและอัปเดตระบบปฏิบัติการและแอปพลิเคชันเพื่อปิดช่องโหว่

3. ติดตั้งและตั้งค่าไฟร์วอลล์: ควบคุมการเชื่อมต่อจากภายนอกให้ปลอดภัย

4. สำรองข้อมูล: สำรองข้อมูลสำคัญเพื่อป้องกันการสูญหาย

5. ควบคุมการเข้าถึง: กำหนดสิทธิ์การเข้าถึงตามความจำเป็น

6. ปิดบริการที่ไม่จำเป็น: ปิดบริการหรือโปรแกรมที่ไม่ใช้งานเพื่อลดช่องโหว่

7. ใช้โปรแกรมป้องกันไวรัส: ติดตั้งและอัปเดตโปรแกรมป้องกันไวรัสและมัลแวร์

-การใช้งาน Firewall และการตั้งค่าเครื่องแม่ข่าย

การใช้งาน Firewall:

1. ติดตั้งและตั้งค่าไฟร์วอลล์เพื่อควบคุมการเข้าถึงจากภายนอก

2. กำหนดกฎการเข้าถึงและบล็อกพอร์ตหรือการเชื่อมต่อที่ไม่จำเป็น

การตั้งค่า เครื่องแม่ข่าย:

1. กำหนดสิทธิ์การเข้าถึงผู้ใช้และอัปเดตระบบปฏิบัติการ

2. ปิดบริการที่ไม่จำเป็นและใช้การเข้ารหัสข้อมูลเพื่อเพิ่มความปลอดภัย

-การป้องกันการโจมตีจากภายนอก (เช่น การใช้งาน SSL, การตั้งค่า VPN)

1. SSL: ใช้ HTTPS แทน HTTP เพื่อเข้ารหัสข้อมูลระหว่างผู้ใช้และเว็บไซต์ ป้องกันการดักจับข้อมูล

2. VPN: ใช้ VPN สร้างการเชื่อมต่อที่ปลอดภัยระหว่างผู้ใช้และเซิร์ฟเวอร์ โดยเข้ารหัสข้อมูล เช่น การเชื่อมต่อระยะไกล

องค์การและการบริหารงานคุณภาพ

1. ความหมายของกลยุทธ์ในองค์การ

องค์การทุกแห่งที่อยู่รอดเติบโตได้นั้น จะต้องสามารถปรับตนเองให้เข้ากับเงื่อนไขสภาพแวดล้อม ที่เปลี่ยนแปลงเสมอ ทั้งนี้ด้วยเหตุผลจากการเกี่ยวข้องกันเป็นเชิงระบบนั่นเอง เพราะขณะใดขณะหนึ่ง ที่องค์การจัดตั้งขึ้นมานั้น สภาพภายในขององค์การจะจัดขึ้นโดยมีความสอดคล้องและเข้าได้กับสภาพแวดล้อม ในขณะนั้นมากที่สุด แต่เมื่อเวลาผ่านไป การเปลี่ยนแปลงของสภาพเศรษฐกิจ สังคม การเมือง และ วิทยาการใหม่ ๆ ที่เกิดขึ้นตลอดเวลาในสภาพแวดล้อมภายนอกนั้น จะส่งผลกระทบถึงสภาพองค์การภายใน ที่เคยจัดไว้นั้นด้วย และการที่องค์การจะสามารถมีประสิทธิภาพต่อไปได้ ก็ต่อเมื่อต้องมีการเปลี่ยนแปลง ให้เข้ากับเงื่อนไขใหม่ได้เสมอ

การบริหารองค์การให้มีประสิทธิภาพจึงมีเรื่องที่สำคัญที่ต้องพิจารณา 2 ประการ คือ

ประการที่ 1 ผู้บริหารต้องเข้าใจถึงลักษณะสภาพแวดล้อมที่เกี่ยวข้อง

ประการที่ 2 ผู้บริหารต้องเข้าใจว่าองค์การสัมพันธ์กับสภาพแวดล้อม อย่างไร โดยพิจารณาในแง่ของการพิจารณาปัจจัยสำคัญ ต่าง ๆ ตามความหมายของพฤติกรรมองค์การ

กลยุทธ์ขององค์การ

การที่องค์การได้แสดงความสัมพันธ์กับสภาพแวดล้อม โดยใช้วิธี การบริหารที่องค์การไปเกี่ยวข้องกับสภาพแวดล้อมที่มีกลุ่มต่าง ๆ ตัวบุคคล องค์การอื่น และสถาบันประเภทอื่น ทั้งหลายที่อยู่ภายในองค์การและนอกองค์การ แต่ในเวลาเดียวกันขณะที่สภาพแวดล้อมได้มีข้อจำกัด ควบคู่มาด้วยนั้น ในสภาพแวดล้อมเองก็ได้เปิดโอกาสต่าง ๆ ขึ้นมาด้วย ทั้งนี้โอกาสต่าง ๆ ที่เปิดขึ้นจะช่วยให้ องค์การสามารถใช้ความสามารถพิเศษของตนที่มีอยู่ และทรัพย์ที่เป็นจุดแข็ง ตลอดจนความคิดความอ่าน ที่จะดำเนินประโยชน์จากสภาพแวดล้อมที่เปิดโอกาสนั้นได้ โดยแสดงมาเป็นภาพได้ดังนี้

2. กลยุทธ์ในฐานะเป็นเครื่องมือสำหรับ ใช้จัดให้สภาพแวดล้อมสอดคล้องกับ ทรัพยากร

กลยุทธ์ หมายถึง กระบวนการที่องค์การพยายามจัดให้มีการใช้ทรัพยากรให้มีประสิทธิภาพ สอดคล้องกับความต้องการที่มีอยู่ และเข้ากันได้กับข้อจำกัดและโอกาสภายในสภาพแวดล้อมที่เกิดขึ้น การจัดให้เข้ากันของทรัพยากรขององค์การกับสภาพแวดล้อมนี้นับว่าเป็นเรื่องสำคัญที่เกี่ยวข้องกับ การพิจารณาขององค์การในแง่ของบทบาทที่องค์การจะแสดงออกภายในสภาพแวดล้อมที่เป็นอยู่ หรือก็คือการกำหนดจุดมุ่งหมายและนโยบายที่เกี่ยวกับความต้องการจะเป็นของบริษัทและธุรกิจที่บริษัทจะดำเนิน นั่นเอง จากที่กล่าวมาถ้าหากนำมากำหนดเป็นกรอบการคิดเกี่ยวกับกลยุทธ์แล้ว จะปรากฏตามรูปแบบต่าง ๆ องค์การจะมีการพิจารณาวิเคราะห์สภาพแวดล้อม เพื่อที่จะตูถึงข้อจำกัดต่าง ๆ ความต้องการและ โอกาสต่าง ๆ ที่มีอยู่ การตรวจสอบสภาพแวดล้อมนี้ในบางครั้งอาจจะมีการวิเคราะห์อย่างละเอียดและ จัดทำเป็นระบบ โดยแสดงเป็นภาพได้ดังนี้

ในขอบเขตความหมายที่กว้างขึ้น กลยุทธ์จะเป็นเครื่องมือในการใช้กำหนดสิ่งที่องค์การต้องการ จะทำให้ได้ ซึ่งผลที่ตามมาภายหลังจากที่ได้มีการพัฒนากลยุทธ์ออกมาแล้ว ก็จะช่วยให้ทุก ๆ กลุ่มที่เกี่ยวข้อง ในองค์การ ทั้งในแง่ตัวบุคคล กลุ่มทำงาน และองค์การโดยรวม สามารถเห็นภาพที่ชัดเจนที่ต่างฝ่ายต่างก็จะ ดำเนินการไปได้ในทิศทางเดียวกัน จากที่กล่าวมาจะเห็นว่ากลยุทธ์และองค์การจะมีการเกี่ยวข้องกันและกัน ผสมผสานกัน ในอันที่จะช่วยชี้ให้เห็นถึงความมีประสิทธิผลขององค์การว่ามีมากน้อยเพียงใด โดยแสดง เป็นภาพได้ดังนี้

3. กลยุทธ์การบริหารที่ทรงประสิทธิภาพ

1. การบริหารต้องเป็นระบบครบทุกด้าน โดยต้องคำนึงถึงทั้งการวางแผน การดำเนินงาน และการควบคุมอย่างมีประสิทธิภาพ

2. การกำหนดแผนงานที่มีประสิทธิภาพ โดยอาศัยเป้าหมายที่ชัดเจนและการมีส่วนร่วมจากพนักงานในการกำหนดเป้าหมายเหล่านั้น

3. การใช้เทคนิคต่าง ๆ เช่น การวัดผล การจูงใจ การควบคุม และการพัฒนา เพื่อเสริมสร้างประสิทธิภาพการทำงานในทุกขั้นตอน

4. การใช้กลยุทธ์ต่าง ๆ เช่น การบริหารโดยเป้าหมาย (MBO), การวางแผนกลยุทธ์ระยะยาว และการสร้างผลสำเร็จร่วมกันระหว่างฝ่ายบริหารและฝ่ายปฏิบัติ

กลยุทธ์เหล่านี้จะช่วยให้เกิดความผูกพันจากพนักงาน เพิ่มขวัญกำลังใจ และกระตุ้นให้เกิดผลผลิตที่สูงขึ้น รวมถึงการปรับตัวให้สอดคล้องกับปัญหาหรือความท้าทายที่จะเกิดขึ้นในอนาคต.

กล่าวโดยสรุป กลยุทธ์การบริหารที่ทรงประสิทธิภาพแต่ละประการตามที่กล่าวข้างต้นนี้อาจกล่าว ได้ว่า ก็คือการบริหารเชิงรวมที่เป็นภารกิจหางการบริหารของผู้บริหารที่มีอยู่เป็นพิเศษ คือ 1 การใช้วิธีบริหารแบบมุ่งหมาย (Goal-Oriented Management) เป็นสำคัญ ทั้งนี้ก็เพราะ ในภาวะเศรษฐกิจที่ประสบปัญหาวิกฤตนั้น ประสิทธิภาพผลผลิตที่ทำได้จะเป็นเรื่องสำคัญยิ่งที่นักบริหาร จะต้องวัดผลและคิดตามตลอดเวลา การตั้งเป้าหมายผลสำเร็จที่ต้องการ และการติดตามผลงานที่ทำได้ ต่างก็ต้องอาศัยเป้าหมายเป็นเครื่องมือที่จะขาดเสียมิได้ 2 การจัดระบบการวางแผนที่สมบูรณ์ ทั้งนี้ก็เพื่อให้องค์การมีความเข้มแข็งจากการมีระบบ การวางแผนกลยุทธ์สำหรับระยะยาว รวมทั้งการมีระบบการวางแผนที่สามารถระบุเป้าหมายผลสำเร็จต่าง ๆ ที่จะสามารถนำมาใช้บริหารงาน เพื่อผลสำเร็จร่วมกันทั้งสองฝ่าย คือ ฝ่ายบริหารและฝ่ายปฏิบัติ 3 การจัดระบบการบริหารงานในขั้นปฏิบัติที่ดีพร้อม นั่นคือการใช้ระบบการวัดผล ประเมินผล การจูงใจทรัพยากรบุคคล ตลอดจนระบบการควบคุม เพื่อใช้เป็นเครื่องมือที่เหมาะสมสำหรับการเสริมสร้าง ประสิทธิภาพผลผลิตให้สูงขึ้นได้ตลอดเวลา

4. ส่วนผสมของกลยุทธ์

การดำเนินตามกระบวนการจัดการเชิงกลยุทธ์ ผู้จัดการต้องออกแบบกลยุทธ์องค์การ (Organizational Strategy) ซึ่งเป็นแผนการปฏิบัติในภาพรวมโดยมีจุดมุ่งหมายเพื่อช่วยให้องค์การบรรลุเป้าหมายต่าง ๆ ในช่วงเวลาที่กำหนด ดังนั้นกลยุทธ์องค์การอาจไม่ใช่เอกสารการวางแผนที่จัดทำขึ้นแล้วนำไปปฏิบัติ แต่กลยุทธ์ต้องมีการปรับปรุงและพัฒนาอย่างต่อเนื่อง เพื่อให้สอดคล้องกับสภาพแวดล้อมปัจจุบัน ต้องมี การผสมผสานระหว่างองค์การที่ท้าทายความสามารถขององค์การในการบรรลุเป้าหมาย กลยุทธ์มีส่วนในการกำหนดทิศทางและกรอบการจัดการให้กับองค์การ ของกลยุทธ์แสดงถึงรายการของสินค้าและบริการที่องค์การนำเสนอต่อลูกค้า มีการกำหนดขอบเขต ในการกำหนดขอบเขตของกลยุทธ์ ทำให้ผู้จัดการใช้ในการเลือกคนขึ้นมา ความสามารถพิเศษขององค์การอันเกิดจากบุคลากรขององค์การที่มีอยู่ ซึ่งสัมพันธ์กับขอบเขตที่องค์การเลือก ความสามารถพิเศษต้องมีความยั่งยืนในระยะยาวจึงจะทำให้องค์การ ดำเนินงานไปได้อย่างมีคุณภาพ ในระดับกลยุทธ์เมื่อองค์การได้กำหนดเป้าหมายและจัดทำแผนในระดับต่าง ๆ ผู้จัดการจำเป็นต้องมีแผนกลยุทธ์ในแต่ละระดับเช่นกัน เพื่อให้เกิดการตัดสินใจและการปฏิบัติเป็นไป ทั่วทั้งองค์การ ดังนั้นกลยุทธ์ขององค์การจึงประกอบด้วย 3 ระดับ ได้แก่

1. กลยุทธ์ระดับองค์การ (Corporate-Level Strategy) คือการกำหนดทิศทางในภาพรวม ขององค์การ กำหนดธุรกิจที่องค์การจะเข้าไปแข่งขันและการจัดสรรทรัพยากรองค์การ กลยุทธ์ระดับองค์การถือเป็นพลังที่ใช้ขับเคลื่อนเป้าหมายและแผนเชิงกลยุทธ์ที่จุดสูงสุดในกระบวนการวางแผนโดยกำหนดแนวทาง อย่างกว้าง ๆ ให้กับผู้จัดการฝ่ายต่าง ๆ ตลอดทั้งองค์การให้ดำเนินตามเพื่อให้บรรลุวัตถุประสงค์ขององค์การนั้นๆ

2. กลยุทธ์ระดับธุรกิจ (Business-Level Strategy) ซึ่งถูกกำหนดโดยผู้บริหารระดับสูง และคณะที่ปรึกษาทางธุรกิจ การดำเนินงานขององค์การวางรากฐานของธุรกิจในอนาคต โดยกลยุทธ์ระดับธุรกิจ อเป็นพลังขับเคลื่อนเป้าหมายและแผนเชิงกลวิธีต่าง ๆ ซึ่งเป็นวิถีทางในการบรรลุเป้าหมายและแผน ชิงกลยุทธ์ในภาพรวมขององค์การในกระบวนการ

3 กลยุทธ์ระดับหน้าที่ (Functional-Level Strategy) กลยุทธ์ระดับนี้สนับสนุนกลยุทธ์ระดับ ธุรกิจ มีกิจกรรมเฉพาะต่าง ๆ ในแต่ละหน้าที่ภายในองค์การ เช่น การตลาด การผลิต การผลิต การเงินและหน้าที่อื่น ๆ กลยุทธ์ระดับหน้าที่เป็นการสร้างให้เกิดข้อได้เปรียบทางการแข่งขัน (Competitive advantage) ซึ่งเป็น สาเหตุให้ธุรกิจประสบความสำเร็จขึ้นอยู่กับคุณค่า (Value) ซึ่งองค์การสามารถสร้างลูกค้าได้ งานของ การสร้างคุณค่าเกิดขึ้นภายในจากหน้าที่ต่าง ๆ ซึ่งอยู่ภายในธุรกิจ หน้าที่เหล่านี้จะต้องเชื่อมโยง จะต้อง สอดคล้องกันเป็นโครงสร้างงาน ซึ่งเรียกว่าเครือข่ายในการสร้างคุณค่า (Value chain) ทุกหน้าที่มีส่วน สร้างคุณค่าและข้อได้เปรียบทางการแข่งขันโดยคำนึงถึงคุณภาพ ประสิทธิภาพและการส่งมอบคุณค่า ให้กับลูกค้าหรือผู้รับบริการ

ดังนั้นการดำเนินกิจการต้องปฏิบัติอย่างสอดคล้องกับกลยุทธ์ระดับองค์การ และกลยุทธ์ระดับธุรกิจ มีการวางแผนกลยุทธ์การดำเนินงานไว้อย่างผสมผสานบทบาทหน้าที่มีความสำคัญในการสร้างงานสร้างความแตกต่างของผลิตภัณฑ์ การทำให้ต้นทุนผลิต หรือต้นทุนในการขาย การบริการต่ำลง ส่งผลให้ธุรกิจมีผลกำไรเพิ่มมากขึ้น การสร้างความจงรักภักดี ให้กับลูกค้าและแสวงหาวิธีการที่มีประสิทธิผลอื่น ๆ ในการดึงดูดให้ลูกค้าอยู่กับเราให้นาน การปรับปรุง พัฒนาผลิตภัณฑ์เป็นสิ่งจำเป็น ผู้จัดการระดับต้น ๆ ต้องพัฒนาส่งมอบคุณค่าที่ดีให้กับลูกค้า

5. กรณีศึกษาการกำหนดกลยุทธ์ อย่างมีเป้าหมายและชัดเจน

กรณีศึกษา การดำเนินงานของ “เซเว่น อีเลฟเว่น” นั้นมีเป้าหมายที่ชัดเจนในการยึดหลักการดำเนิน ธุรกิจด้วยความจริงใจ ซื่อสัตย์สุจริต และไม่หวังผลกำไรเกินควร อีกทั้งดำรงความเป็นองค์กรที่มีส่วนร่วม ในการพัฒนาวิถีไทยให้ดำเนินไปอย่างมีคุณค่า ซึ่งนำมาสู่วิสัยทัศน์ขององค์การ คือ “เราให้บริการความสะดวก กับทุกชุมชน” และปรัชญาขององค์การคือ “เราปรารถนารอยยิ้มจากลูกค้าด้วยทีมงานที่มีความสุข” เพื่อเดินไปสู่พันธกิจที่วางไว้ว่า “มุ่งสร้างความผูกพันกับลูกค้า ด้วยการสรรหาสินค้าและบริการที่มีคุณภาพ ภายใต้การบริหารงานแบบ Harmony พร้อมทั้งสร้างสัมพันธภาพที่ดีกับสังคมและชุมชน”ดังนั้นคณะผู้บริหารและพนักงานทุกคนของเซเว่น อีเลฟเว่นจึงมุ่งมั่นค้นหาและตอบสนอง ความต้องการของลูกค้าที่เปลี่ยนแปลงตลอดเวลาทั้งในปัจจุบันและอนาคต โดยเน้นความสะดวกซื้อและบริการที่เป็นเลิศ ด้วยเครือข่ายที่ครอบคลุมและมีประสิทธิภาพของพันธมิตร

งหนึ่งที่เซเว่น อีเลฟเว่นนั้นได้รับการยอมรับกันมากในวงการธุรกิจก็คือ “ความเป็นเลิศในด้าน การมีภาวะผู้นำ” ความสำเร็จส่วนนี้จะเกิดขึ้นเองไม่ได้ ถ้าขาดผู้บริหารที่มีวิสัยทัศน์อันกว้างไกล และให้ความสำคัญกับการพัฒนาบุคลากรควบคู่ไปกับการพัฒนาองค์การมาโดยตลอด ซึ่งมาจากการ สร้างค่านิยม 11 ประการ เพื่อเป็นแนวทางในการสร้างคนที่มีคุณภาพ ซึ่งได้แก่

1. มีความจริงใจ ไม่หลอกใช้ผู้อื่น

2. ไม่ศักดินา ไม่แบ่งชั้นวรรณะ ไม่ยกตนข่มท่าน ไม่แบ่งแยกสถาบัน

3. ใช้ปิยวาจา เป็นผู้ฟังที่ดี ใช้คำพูดในเชิงสร้างสรรค์

4. อย่าหลงอำนาจ ไม่ยึดอำนาจ ตำแหน่งหน้าที่ ไม่ข่มเหงผู้น้อย

5. เป็นแบบอย่างที่ดี มีความขยัน มีวินัยในตนเอง และทัศนคติที่ดีต่อการทำงาน

6. มีความยุติธรรม ไม่ฟังความข้างเดียว ไม่หลงเชื่อคนง่าย

7. ให้ความเมตตา ให้ความช่วยเหลือลูกน้องด้วยความเต็มใจ

8. กล้าตัดสินใจ ตัดสินใจบนพื้นฐานของหลักการข้อเท็จจริงและประสบการณ์

9. อาทรสังคม ส่งเสริมสังคมด้วยความรับผิดชอบ

10. บ่มเพาะคนดี สร้างคนมีความสามารถให้กับองค์การ

11. มีใจเปิดกว้าง รับฟังความเห็นของผู้ร่วมงานทุกระดับ เปิดใจรับแนวคิดใหม่ ๆ ของโลก

6. ประสิทธิภาพของการทำงานอาชีวศึกษา และตลาดแรงงานในประเทศไทย

ในการศึกษาความสัมพันธ์ระหว่างการเรียนสายอาชีวศึกษาและสถานภาพการทำงาน ได้แบ่ง สถานภาพการทำงานออกเป็น 5 ประเภท ได้แก่ นายจ้าง ผู้ที่ประกอบธุรกิจส่วนตัวโดยไม่มีลูกจ้าง ลูกจ้าง เอกชน ผู้ที่ทำงานให้ครอบครัวโดยไม่ได้รับค่าจ้างและอื่น ๆ (ซึ่งรวมลูกจ้างรัฐบาล ลูกจ้างรัฐวิสาหกิจ ผู้รับจ้างทำงานหลายเจ้าและการรวมกลุ่ม) ผลการศึกษาพบว่า ผู้ที่เรียนสายอาชีวศึกษามีโอกาสทำงาน ในสถานะผู้ที่ประกอบธุรกิจส่วนตัวโดยไม่มีลูกจ้างและผู้ที่ทำงานให้ครอบครัวโดยไม่ได้รับค่าจ้างน้อยกว่า การทำงานในสถานะอื่น ๆ สำหรับประเภทของงาน (ซึ่งแบ่งเป็นภาคการผลิตขั้นพื้นฐาน ภาคอุตสาหกรรม และภาคบริการ) ผลการศึกษาพบว่า สำหรับผู้ชายที่เรียนสายอาชีวศึกษานั้น มีโอกาสที่จะทำงานภาค การผลิตขั้นพื้นฐานน้อยกว่าการท้างานในภาคบริการ ในส่วนของผู้หญิงที่เรียนสายอาชีวศึกษา มีโอกาส ที่จะทำงานภาคการผลิตขั้นพื้นฐานและภาคอุตสาหกรรมน้อยกว่าการทำงานในภาคบริการ ทั้งนี้ผลการศึกษา สอดคล้องกับวัตถุประสงค์ของการศึกษาในสายอาชีพเพื่อฝึกทักษะที่เกี่ยวข้องกับการจ้างงาน โดยเฉพาะ สำหรับภาคอุตสาหกรรมและภาคบริการ

ระดับทักษะในการทำงาน แบ่งเป็นงานที่ใช้ทักษะขั้นสูง (high skilled) งานที่ใช้ทักษะด้านฝีมือหรือแรงงาน (skilled manual) งานที่ใช้ทักษะที่ไม่ใช่ด้านฝีมือหรือแรงงาน (skilled non-manual) และ งานที่ไม่ใช้ทักษะ (low skilled) ซึ่งผลการศึกษาพบว่าผู้ชายที่เรียนด้านอาชีวศึกษามีแนวโน้มที่จะทำงาน ที่ใช้ทักษะขั้นสูง งานที่ใช้ทักษะด้านฝีมือหรือแรงงาน งานที่ใช้ทักษะที่ไม่ใช่ด้านฝีมือหรือแรงงานมากกว่า การทำงานที่ไม่ใช้ทักษะ โนขณะที่ผู้หญิงที่เรียนด้านอาชีวศึกษามีแนวโน้มที่จะทำงานที่ใช้ทักษะขั้นสูงและ งานที่ใช้ทักษะด้านที่ไม่ใช่ด้านฝีมือหรือแรงงานมากกว่างานที่ไม่ใช้ทักษะและงานที่ใช้ทักษะด้านฝีมือหรือแรงงาน

จากการศึกษานี้พบว่า การเรียนอาชีวศึกษาเป็นการเตรียมบุคคลเข้าสู่ตลาดแรงงานบางประเภท ที่ใช้ทักษะและข้อได้เปรียบนี้ส่งผลให้ผู้ที่เรียนอาชีวศึกษามีรายได้สูงกว่าผู้ที่เรียนจบชั้นมัธยมศึกษาปีที่ 6 สายสามัญ ซึ่งผลการศึกษานี้สนับสนุนให้มีการพัฒนาอาชีวศึกษาในประเทศไทยให้มีคุณภาพและเป็นที่ นิยมมากขึ้น (ที่มา : จากหนังสือพิมพ์ฐานเศรษฐกิจ ปีที่ 37 ฉบับที่ 3,323 วันที่ 17-20 ธันวาคม พ.ศ. 2560)

7. กลยุทธ์ค่าตอบแทน

กลยุทธ์ค่าตอบแทนที่เป็นธรรมกับทุกฝ่าย องค์การจะจ่ายค่าตอบแทนระบบใดขึ้นอยู่กับ ขีดความสามารถและศักยภาพที่แท้จริงขององค์การ ซึ่งแตกต่างกันไปแต่ละองค์การ อย่างไรก็ตามการจัดระบบ ค่าตอบแทนที่ดีต้องไม่สร้างความเสียหายให้กับผู้ปฏิบัติงาน องค์การต้องพยายามทำให้เกิดความพึงพอใจ กับพนักงานให้มากที่สุด เพราะการจ่ายค่าตอบแทนที่เป็นธรรม เป็นการรักษาดุลยภาพของผู้เกี่ยวข้อง กลยุทธ์การจ่ายค่าตอบแทน มีดังนี้

1.การจ่ายค่าตอบแทนต้องให้เกิดความเป็นธรรมภายในองค์การ หมายความว่า ลักษณะงาน ที่แตกต่างกันระบบการจ่ายค่าตอบแทนอาจจะแตกต่างกันไป งานใดมีความสำคัญและสร้างมูลค่าตอบแทน ในสาขาอาชีพที่ขาดแคลน หากสาขาอาชีพใดสำคัญกับองค์การหรือขาดแคลน องค์การก็จ่ายค่าตอบแทน ให้สูงขึ้น เพื่อลดปัญหาการขาดแคลนในสาขาอาชีพนั้น ๆ

2.การจ่ายค่าตอบแทนต้องเทียบเคียงกับภายนอกได้ จ่ายให้ระดับสูงกว่าองค์การอื่น ๆ หากองค์การ ต้องการสร้างภาพลักษณ์ที่เหนือกว่าองค์การอื่น ๆ หรือต้องการคัดเลือกบุคลากรที่มีความสามารถสูงกว่าทั่วไป หรือเน้นที่ตัวเงินไม่เน้นการให้สวัสดิการ หรือเนื้องานมีมากกว่าในตำแหน่งเดียวกัน องค์การก็อาจใช้วิธีการจ่าย ให้สูงกว่าองค์การอื่น ๆ หรือจ่ายให้ระดับเท่ากันกับองค์การอื่น ๆ องค์การอาจไม่สามารถจะจ่ายให้ต่ำกว่าได้ หรือไม่สามารถจ่ายให้สูงกว่าได้ เนื่องจากเหตุผลต่าง ๆ องค์การจึงต้องจ่ายให้ในระดับมาตรฐานทั่ว ๆ ไป เพื่อคงระดับของการแข่งขันเอาไว้

3.การจ่ายค่าตอบแทนต้องเป็นธรรมกับพนักงาน

4.การจ่ายค่าตอบแทนต้องให้สอดคล้องกับศักยภาพขององค์การ ควรมีค่าตอบแทนอื่น ๆ เช่น ประโยชน์และบริการที่พนักงานได้รับนอกเหนือไปจากเงินเดือนและค่าจ้าง เช่น ประกันสังคม ค่ารักษา พยาบาล การลาโดยได้รับค่าจ้าง การประกันสุขภาพและชีวิต ค่าขนย้าย เงินช่วยเหลือการศึกษาบุตร

รางวัลตอบแทนด้านอาชีพ (Career Reward) เป็นรางวัลตอบแทนให้พนักงานเพื่อให้มีอาชีพที่มี ความมั่นคง มีโอกาสก้าวหน้า พัฒนาตนเองและมีความยืดหยุ่นในอาชีพที่อาจเปลี่ยนแปลงได้เมื่อต้องการ

รางวัลตอบแทนด้านสังคม (Social Reward) เป็นรางวัลตอบแทนโนด้านสถานภาพ และสภาพแวดล้อมในการทำงาน เช่น การให้สัญลักษณ์การแสดงสถานภาพ การให้การยกย่องชมเชย ผลการทำงาน ความสนุกสนานในการทำงาน การมีเพื่อนร่วมงานที่ดี

สังคมไทยในยุคดิจิทัล

1. ความหมายของยุคดิจิทัล (Digital Era)

ยุคดิจิทัล (Digital Era) คือ ยุคของอิเล็กทรอนิกส์ทั้งด้านที่เกี่ยวข้องกับเทคโนโลยีที่ใช้ความรวดเร็ว ในการสื่อสาร ข้อมูล ความรู้ต่าง ๆ เช่น ข่าว ภาพ วิดีโอ ทำให้ทุกคนสามารถเข้าถึงได้อย่างรวดเร็ว ทุกที่ ทุกเวลานับตั้งแต่การเกิดระบบอินเทอร์เน็ตและพัฒนาการของเทคโนโลยีด้านการสื่อสารและระบบ คอมพิวเตอร์ ทำให้สังคมออนไลน์เข้ามาอยู่ในชีวิตประจำวันของคนทั่วโลก การถ่ายโอนกิจวัตรประจำวัน ของคนทั่วโลกไปสู่สังคมออนไลน์ก็มีมากขึ้นตามลำดับ ทำให้คนที่อยู่ในทุกมุมโลกสามารถเชื่อมหากันได้ง่าย สังคมออนไลน์จึงมีความสำคัญควบคู่ไปกับสังคมปกติในชีวิตประจำวันของคนทั่วโลก

2. ประเภทของเครือข่ายสังคมออนไลน์

ประเภทของเครือข่ายสังคมออนไลน์ที่นิยมใช้ในปัจจุบันมีดังนี้

ประเภทที่ 1 เครือข่ายสังคมออนไลน์ที่เน้นการสร้างเครือข่ายทางสังคมในอินเทอร์เน็ต และเชื่อมโยงเป็นกลุ่มบุคคลเพื่อติดต่อสื่อสารหรือติดตามข้อมูลระหว่างกัน

สำหรับเครือข่ายสังคมออนไลน์ที่เป็นที่นิยมของคนไทยและใช้กันมากในชีวิตประจำวันมีดังนี้

1. Facebook เป็นสื่อสังคมออนไลน์หลักของคนไทย จากข้อมูลของ The Web Flight Co., Ltd (TWF Agency) พบว่า คนไทยมีบัญชีเฟซบุ๊กราว 51 ล้านบัญชี (84.9 ของจำนวนประชากรอายุ 13 ปีขึ้นไป) โดยเป็นเครือข่ายสังคมออนไลน์ที่เปิด โอกาสให้สมาชิกสร้างข้อมูลส่วนตัว (Profile) และการติดตามเครือข่ายสังคมออนไลน์ ในลักษณะของเพื่อน (Frend) กลุ่มที่สนใจ (Group) โดยที่สมาชิกสามารถโพสต์ข้อความ ภาพ วิดีโอเพื่อให้สมาชิกในเครือข่ายสังคมออนไลน์ทราบ ตลอดจนทราบการเคลื่อนไหวของเพื่อนหรือกลุ่มที่ ตนติดตามได้ นอกจากนี้ยังสามารถส่งต่อ (Share) เพื่อขยายการรับรู้ข้อมูลของตนไปยังเพื่อนหรือกลุ่มได้

2. Instagram (IG) เป็นสื่อสังคมออนไลน์หลักของคนไทยอีกเครือข่ายหนึ่ง จากข้อมูลของ The Web Flight Co., Ltd (TWF Agency) พบว่า คนไทยมีบัญชี อินสตาแกรมราว 16 ล้านบัญชี (26.6 ของจำนวนประชากรอายุ 13 ปีขึ้นไป) โดยที่ อินสตาแกรมมีการติดตามเพื่อนเช่นเดียวกับเฟซบุ๊ก แต่เน้นการโพสต์ภาพและวิดีโอพร้อมคำบรรยายภาพ ไม่สามารถโพสต์ข้อความอย่างเดียวเหมือนเฟชบุ๊กได้ มีฟังก์ชันการตกแต่งรูป ให้น่าสนใจ

3. X หรือที่รู้จักกันในชื่อเดิมว่า Twitter เป็นสื่อสังคมออนไลน์บนแพลตฟอร์ม โซเชียลมีเดียที่ผู้ใช้สามารถโพสต์ข้อความสั้น ๆ ที่เรียกว่า หวีด ความยาวสูงสุด 280 ตัวอักษร ทวีดเหล่านี้สามารถประกอบด้วยข้อความ รูปภาพ วิดีโอ และลิงก์ ผู้ใช้สามารถ ติดตาม ผู้ใช้รายอื่นเพื่อดูทวีตของพวกเขา โต้ตอบ กับทวีตโดย กดไลก์ คอมเมนต์ และรีทวีต ค้นหา เนื้อหาที่เกี่ยวข้องกับหัวข้อที่สนใจ สร้างกลุ่ม ส่งข้อความส่วนตัว เข้าร่วมแฮชแท็ก และอื่น ๆ อีกมากมาย

ประเภทที่ 2 เครือข่ายสังคมออนไลน์ที่เน้นการแบ่งปันระหว่างกันในบทบาทของผู้นำเข้าข้อมูล หรือผลตอบแทนจากเว็บไซต์ และเจ้าของเว็บไซต์อาจได้รับผลตอบแทนจากการโฆษณาผลิตภัณฑ์ต่าง ๆ

ประเภทที่ 3 เครือข่ายสังคมออนไลน์ที่เน้นการถ่ายทอดเรื่องราว หรือข้อความแก่สมาชิกหรือบุคคลทั่วไป เรียกว่า บล็อก (Blog) หรือ เว็บปล็อก (Weblog) คือ สื่อส่วนบุคคลที่เรียกว่า บล็อกเกอร์ (Blogger) ใช้เผยแพร่ข้อมูล ข่าวสาร ข้อคิดเห็นส่วนตัวของผู้นั้น และสามารถแบ่งปันให้บุคคลอื่น ๆ เข้าไปอ่าน หรือแสดงความคิดเห็นเพิ่มเติมได้ การแสดงเนื้อหาของบล็อกจะเรียงลำดับ ตามห้วงเวลา กรณีการเขียนข้อความสั้น ๆ เพื่อแสดงสถานะตนเองว่าทำอะไร หรือแจ้งข่าวสารที่มีใจความสำคัญแก่คนในกลุ่มสังคมออนไลน์ด้วยกัน เรียกว่า ไมโครบล็อก (Micro Blog)

ประเภทที่ 4 เครือข่ายสังคมออนไลน์ที่เน้นการระดมความร่วมมือของสมาชิกที่สนใจ ในสิ่งเดียวกันเพื่อผลักดันประเด็นหนึ่ง ๆ ให้ประสบความสำเร็จ ทั้งการระดมกำลัง ทั้งความคิด กำลังทรัพย์ หรือที่เรียกว่า Crowdsourcing

ประเภทที่ 5 เครือข่ายสังคมออนไลน์ที่เน้นการสร้างสถานการณ์ บนโลกออนไลน์เพื่อให้สมาชิกหรือผู้ใช้บริการเข้าสื่อสารหรือกระทำการระหว่างกัน บนอินเทอร์เน็ตในลักษณะโลกเสมือนจริง (Virtual Reality/Virtual Worlds) ในรูปแบบที่เรียกว่า “จักรวาลนฤมิต (Metaverse)” โดยการสร้างสภาพแวดล้อม ของโลกแห่งความจริงและเทคโนโลยีเข้าด้วยกัน ด้วยเทคโนโลยีผสานโลกแห่งความจริง และวัตถุต่าง ๆ เข้าด้วยกัน (Augmented Reality : AR) และการจำลองภาพ ให้เสมือนจริงแบบ 360 องศา (Virtual Reality : VR)

ประเภทที่ 6 เครือข่ายสังคมออนไลน์ที่เน้นลักษณะของพื้นที่ ใช้สำหรับการแลกเปลี่ยน บทสนทนา การพูดคุย การอภิปรายในสังคมออนไลน์

ประเภทที่ 7 เครือข่ายสังคมออนไลน์ที่เน้นการสร้างสื่อกลางแลกเปลี่ยนสินค้าหรือบริการ ในลักษณะสกุลเงินดิจิทัล (Cryptocurrency) โดยอาศัยกลไกทางคณิตศาสตร์เพื่อกำหนดจำนวน และเทคโนโลยีบล็อกเซน (Blockchain) เป็นตัวช่วยด้านความปลอดภัย (Security) และความน่าเชื่อถือ (Trust) ในการทำธุรกรรมการเงินโดยไม่ต้องอาศัยคนกลาง ผู้คนส่วนหนึ่งใช้สกุลเงินดิจิทัลเพื่อการ แลกเปลี่ยนบนโลกออนไลน์ บางส่วนถือครองไว้สำหรับการเก็งกำไรในภาวะของความผันผวนของค่าเงิน สกุลต่าง ๆ

ประโยชน์และข้อจํากัดของเครือข่ายสังคมออนไลน์

สังคมออนไลน์มีประโยชน์ในชีวิตประจำวันและการทำงานดังนี้

* เป็นพื้นที่ที่ทำให้บุคคลสามารถแลกเปลี่ยนข้อมูลซึ่งกันและกันโดยไม่จำกัด ด้านระยะทางและสถานที่ ขอบลง

* เป็นช่องทางการติดต่อสื่อสารแนวใหม่ทั้งของภาครัฐและภาคเอกชน เนื่องจากลดภาร ด้านบุคลากรและค่าใช้จ่ายต่าง ๆ ตลอดจนสื่อสารได้เป็นวงกว้าง

* เป็นคลังข้อมูลความรู้จำนวนมหาศาลเพื่อตอบสนองความต้องการของบุคคล ในทางกลับกั เป็นช่องทางการเผยแพร่ความรู้ของบุคคลสู่สังคมเป็นวงกว้าง

* ช่วยลดค่าใช้จ่ายในด้านต่าง ๆ เช่น

- ด้านการจัดทำเอกสาร การถ่ายสำเนา โดยใช้การจัดทำเป็นเอกสารอิเล็กทรอนิกส์ (E-document) แทน

- ด้านสถานที่ เช่น จากการใช้ห้องประชุม การเช่าห้องประชุม ห้องเรียน มาเป็นการใช้ห้อง ทางออนไลน์แทน

- ด้านค่าธรรมเนียม เช่น การชำระ ค่าสาธารณูปโภคต่าง ๆ บุคคลสามารถทำได้เอง ผ่านระบบออนไลน์

* สร้างความสะดวกสบายมากกว่าในอดีต

ทั้งนี้เนื่องจากการเข้าถึงสังคมออนไลน์ได้จะต้องประกอบด้วยองค์ประกอบหลัก ดังนี้

* ด้านบุคคล

บุคคลต้องมีความรู้ในเรื่องการใช้อุปกรณ์และโปรแกรมการใช้งานและการเข้าถึงสือสังคม ออนไลน์

* ด้านอุปกรณ์

อุปกรณ์คอมพิวเตอร์จะต้องรองรับการเข้าระบบอินเทอร์เน็ตและการติดตั้งโปรแกรม เพื่อการใช้งานสื่อสังคมออนไลน์ได้ ไม่ว่าจะอยู่ในรูปของคอมพิวเตอร์ตั้งโต๊ะ โน้ตบุ๊ก โทรศัพท์มือถือ หรืออุปกรณ์อื่นใดที่สามารถประมวลผลได้ในลักษณะเดียวกัน

* ด้านโครงสร้างพื้นฐาน

ได้แก่ ระบบไฟฟ้าและระบบอินเทอร์เน็ตจะต้องมีและเพียงพอต่อการเข้าใช้สื่อสังคม

* ด้านโปรแกรม

ต้องมีโปรแกรมหรือแอปพลิเคชันที่ติดตั้งในอุปกรณ์คอมพิวเตอร์

* ด้านนโยบายภาครัฐ

นโยบายภาครัฐจะต้องไม่ปิดกั้นสื่อสังคมออนไลน์ ในทางกลับกันหากรัฐเห็นว่าสื่อสังคม ออนไลน์นั้นกระทบกับความมั่นคง ความสงบเรียบร้อย หรือศีลธรรมอันดีของบว่าสือสัมแล้ว ก็อาจจะปิดกั้นการเข้าถึงสื่อสังคมออนไลน์หนึ่ง ๆ ได้

3. สถานการณ์ของสังคมไทยและสังคมโลกยุคดิจิทัล

ปัจจุบันโลกออนไลน์เข้ามาสู่ชีวิตประจำวันของคนเราจนกลายเป็นเรื่องธรรมดา ส่วนหนึ่ง ก็เพราะว่าสร้างความสะดวกสบาย และอาจช่วยลดภาระค่าใช้จ่ายด้านต่าง ๆ ได้ ซึ่งมีการใช้ประโยชน์ ในการสื่อสารยุคดิจิทัลดังต่อไปนี้

* ด้าานการติดต่อสื่อสาร

การติดต่อสื่อสารเป็นวัตถุประสงค์ประการแรกสุดในการจัดให้มีระบบอินเทอร์เน็ตและ การสร้างสังคมหรือโลกออนไลน์ขึ้นมา ในปัจจุบันมีการใช้ประโยชน์ในชีวิตประจำวัน ดังนี้

- จดหมายอิเล็กทรอนิกส์ (E-mail)

- แชด (Chat)

- การประชุมผ่านสื่ออิเล็กทรอนิกส์ (Video Conference)

- ประกาศข้อความหรือโพสต์ (Post)

- การอัปโหลด (Upload)

* ด้านการศึกษา

การสื่อสารในยุคดิจิทัลก่อให้เกิดประโยชน์ในด้านการศึกษาเป็นอย่างมาก ทั้งนี้เนื่องจาก ผู้ส่งสาร โดยเฉพาะอย่างยิ่งครูอาจารย์สามารถนำเทคโนโลยีทางคอมพิวเตอร์มาใช้สำหรับการจัด การเรียนการสอนได้ เช่น การจัดการเรียนการสอนออนไลน์ การให้นักเรียนไปศึกษาความรู้เพิ่มเติม จากข้อมูลในอินเทอร์เน็ต การใช้โปรแกรมสื่อสารสอนด้วยเทคนิคใหม่ ๆ การจัดทำห้องเรียนออนไลน์ สำหรับการใส่ใบความรู้ให้นักเรียนได้อ่าน การส่งการบ้านทางออนไลน์

* ด้านเศรษฐกิจ

มีการใช้การสื่อสารในยุคดิจิทัลมาใช้ประโยชน์ในด้านเศรษฐกิจหลายลักษณะ เช่น ด้านพาณิชยกรรม จากอดีตเป็นการซื้อขายของกันที่หน้าร้าน ปรับเปลี่ยนมาเป็นการสั่งสินค้า ผ่านระบบตลาดออนไลน์ (Online Market)

* ด้านการบริการ

มีแอปพลิเคชันช่วยเพิ่มประสิทธิภาพการให้บริการเพิ่มมากขึ้น เช่น การจอง โรงแรม การจองเที่ยวบิน ช่วยเพิ่มความสะดวกแก่ผู้ใช้บริการ

* ด้านการธนาคาร

ลูกค้าธนาคารไม่จำเป็นต้องไปติดต่อธนาคารเพื่อทำธุรกรรมทางการเงิน เช่น การฝากเงิน การโอนเงิน การชำระค่าบริการต่าง ๆ แต่สามารถปรับเปลี่ยนไปในรูปแบบของการ ทำธุรกรรมธนาคารบนอินเทอร์เน็ต (Internet Banking)

* การติดต่อภาครัฐ

นอกจากนี้ระบบการสื่อสารยุคดิจิทัลยังมีส่วนอำนวยความสะดวกในการติดต่อภาครัฐ ไม่จำเป็นต้องเดินทางไปยังหน่วยงานต่าง ๆ และภาครัฐสามารถใช้ช่องทางเหล่านั้นติดต่อกับประชาชน ได้เป็นวงกว้าง เช่น ด้านสาธารณสุข การยื่นภาษีออนไลน์ การจองคิวทำใบขับขี่กับกรมการขนส่งทางบก

สรุป

การสื่อสารคือปัจจัยพื้นฐานสำคัญอย่างหนึ่งที่ทำให้มนุษย์สามารถอยู่ร่วมกัน เป็นสังคมจนถึงทุกวันนี้ คนเราไม่ว่าจะมีอายุ เพศ หรือบทบาทใดในสังคมก็ตาม ล้วนต้องพึ่งพาการสื่อสารและการมีปฏิสัมพันธ์ระหว่างบุคคลอื่นอยู่เสมอ ไม่ว่าจะเป็น บุคคลในครอบครัว เพื่อน คนที่ทำงาน หรือผู้คนที่พบเจอในแต่ละวันเพื่อทำ กิจกรรมต่าง ๆ เช่น การซื้อของ การเดินทาง การเล่นกีฬา และเมื่อโลกาภิวัตน ทำให้สังคมเปลี่ยนแปลงมาอยู่ในรูปแบบของโลกไร้พรมแดน การสื่อสารจึงมีรูปแบบ ที่แตกต่างจากในอดีต หากมองไปรอบตัวจะพบว่าปัจจุบันการสื่อสารผ่านกระดาษ และน้าหมึก อาจไม่จำเป็นอีกต่อไป เมื่อมนุษย์ยุคดิจิทัลสามารถสื่อสารผ่านระบบ ไร้สายหรืออินเทอร์เน็ตที่สามารถเกิดขึ้นได้ทุกที่ ทุกเวลา และทุกสถานที่ ความสะดวก สบายของการสื่อสารรูปแบบนี้ จึงได้นำพาผู้คนทั้งที่รู้จักและไม่รู้จักกันจากทั่วทุกมุมโลก มาอยู่ร่วมกัน ซึ่งก่อให้เกิดเครือข่ายสังคมขนาดใหญ่ ดังนั้นเราทุกคนที่ใช้ชีวิตใน สังคมออนไลน์ จึงจำเป็นต้องเรียนรู้มารยาทในการสื่อสารยุคดิจิทัลเพื่อให้การมี ปฏิสัมพันธ์ผ่านหน้าจอเป็นไปด้วยความสงบเรียบร้อย เกิดความเข้าใจอันดีซึ่งกันและกัน บรรลุวัตถุประสงค์ของการสื่อสารและเกิดประโยชน์ต่อตนเองและผู้อื่น
